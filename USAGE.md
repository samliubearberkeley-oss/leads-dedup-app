# Leads 去重系统 - 使用指南

## ✅ 已完成功能

### 1. 核心功能
- ✅ **自动去重** - 上传CSV后自动识别已发送的leads
- ✅ **数据持久化** - 所有数据存储在InsForge数据库中
- ✅ **递归去重** - 每次上传都会自动过滤已存在的记录
- ✅ **批量操作** - 支持复制全部邮箱、导出CSV

### 2. 表格显示（重点字段）
表格重点显示三个关键字段：
- 📺 **YouTube Channel** - 频道名称（可点击跳转）
- 👤 **名字/用户名** - 显示用户名或频道名
- 📧 **邮箱** - 邮箱地址（可一键复制）

### 3. 搜索功能
- ✅ 实时搜索频道、用户名、邮箱
- ✅ 显示搜索结果统计
- ✅ 一键清除搜索

### 4. 数据管理
- ✅ 查看已发送的leads（364条已导入）
- ✅ 查看新leads（未发送的）
- ✅ 删除单个记录
- ✅ 复制单个或全部邮箱

## 📋 使用流程

### 步骤1: 上传CSV文件
1. 点击"上传/粘贴"标签
2. 拖拽CSV文件或点击选择文件
3. 支持多文件同时上传

### 步骤2: 查看处理结果
- 系统自动处理并显示：
  - ✅ 新leads数量（未发送的）
  - ❌ 重复leads数量（已发送的）

### 步骤3: 查看新Leads
1. 点击"新Leads"标签
2. 查看未发送的联系人列表
3. 表格显示：YouTube Channel、名字、邮箱

### 步骤4: 保存到数据库
1. 确认新leads无误
2. 点击"保存到数据库"按钮
3. 保存后的leads会自动出现在"已发送"列表中

### 步骤5: 管理已发送Leads
1. 点击"已发送"标签
2. 使用搜索框快速查找
3. 可以删除不需要的记录
4. 支持复制单个或全部邮箱

## 🔍 搜索功能

在"已发送"或"新Leads"页面：
- 输入关键词搜索频道、用户名或邮箱
- 实时显示匹配结果
- 显示"X / Y 条结果"

## 📊 数据统计

页面顶部显示：
- **364 已发送** - 数据库中已保存的leads总数
- **362 邮箱** - 唯一邮箱数量

## 💡 使用技巧

1. **批量复制邮箱**
   - 在"已发送"或"新Leads"页面
   - 点击"复制全部邮箱"按钮
   - 所有邮箱会复制到剪贴板（每行一个）

2. **快速查找**
   - 使用搜索框输入关键词
   - 支持搜索频道名、用户名、邮箱

3. **导出数据**
   - 在"新Leads"页面
   - 点击"导出CSV"按钮
   - 下载包含所有新leads的CSV文件

4. **删除记录**
   - 在"已发送"页面
   - 点击每行右侧的删除按钮
   - 确认后删除

## ⚠️ 注意事项

- CSV文件必须包含`URL`列（必需）
- 邮箱字段为空时显示"无邮箱"
- 已保存的数据会永久存储，请谨慎删除
- 建议定期备份数据库

## 🚀 启动应用

```bash
cd leads-dedup-app
npm run dev
```

应用会在 `http://localhost:5173/` 启动

## 📝 CSV格式要求

CSV文件应包含以下列（不区分大小写）：
- `URL` - YouTube频道URL（必需）
- `CHANNEL` - 频道名称
- `USERNAME` - 用户名
- `EMAIL` - 邮箱地址
- `VALID EMAIL` - 验证过的邮箱

其他可选字段：POST PER MONTH, SIMILARITY, COUNTRY, SUBSCRIBERS, TOPICS, AUDIENCES等
